## Başlarken

Projenin klasör yapısı:

```
	website
	├── README.md
	├── package.json
	├── .gitignore
    ├── components
	│   ├── GlobalStyle.js
	│   ├── Icon.js
	│   └── Social.js
	├── pages
	│	├── _app.js
	│	├── _document.js
	│	└── index.js
    └── utils
        └── selection.json
```

### Kullanılan Araçlar

- **Next.js** - Projenin temelidir.
- **Styled Components** - JS içine CSS eklemek için kullanılmıştır.
- **Icomoon** - Icon yapısı için kullanılmıştır.

### Kurulum

Paketlerin kurulumu için;

```
  npm install
```
